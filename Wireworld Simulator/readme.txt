In the final submission directory, 2 directories can be found:

1) Wireworld GUI: This directory contains the source code of the project (Wireworld GUI/src) and 
the meta-info provided by the IDE (Intellij IDEA) it was developed in.

2) Wireworld Executable: This directory contains a java executable for easily running the software,
as well as the images the code accesses. Any save files made in the software will be stored in this directory.
